# Web front end (placeholder)

Reserved home for the delivery surface: the web app Jack uses to interact with
the data and the AI products. Built in Phase 5, once a real feature is ready to
expose.

Planned:
- Google OAuth (OIDC) sign-in with an email allowlist gating every data/model endpoint.
- Reuses `src/diskos/` (io, palyno, welllog, llm) directly as the backend.
- Surfaces the explorer (maps, imagery, plots), search, and the wiki assistant.
- Likely hosted on Modal.

Framework is chosen in Phase 5, not before.
